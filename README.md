# FOPPRplot
FOPPR data operate &amp; plot online by PyQt5,matplotlib,pandas programing with python3

# Download
`git clone https://github.com/curtis992250/FOPPRplot`

# Setup
```command
cd FOPPRplot
mkdir output
pip install -r requirement.txt
```

# Usage
`python main.py`
